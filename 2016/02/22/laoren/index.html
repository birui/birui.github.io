文章可以拥有如下属性：

Setting Description Default

layout Layout post或page

title 文章的标题

date 创建日期 文件的创建日期

updated 修改日期 文件的修改日期

comments 是否开启评论 true

tags 标签

categories 分类

permalink url中的名字 文件名

动态博客中通过发布文章页面设置的各种属性，在hexo里要这样设置。

分类和标签
例如：

===========

categories:

  • 日记

tags:

  • Hexo

  • node.js

===========
